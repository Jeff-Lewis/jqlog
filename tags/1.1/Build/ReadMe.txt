jqlog X.X

jqlog is a logging framework plugin for jQuery that can be configured and extended allowing you to "roll your own" logging framework.  For more information visit http://code.google.com/p/jqlog/

jquery.jqlog-1.1.custom.js a custom version of jqlog combining the following plugins and targets.

jquery.jqlog-X.X.js
jqlog.levels-X.X.js
jqlog.cookie-X.X.js
jqlog.target.console-X.X.js
